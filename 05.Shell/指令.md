# 打包相关指令

下载yum源到本地，不安装

```shell
yum install --downloadonly --downloaddir=/download ricci
```

编译

```
gzexe file      #文件加密
gzexe -d file   #文件解密
```

查找

```shell
grep 查找文件内容
find 查抄文件
grep -e 根据正则表达式查找文件内容
grep "a" /.in 
grep -i "" 文件名 不区分大小写
grep -v "" 文件名
# 查找指定文件后，在文件里查找内容
find / -type f -rn -name "*.log"|xargs grep "ERROR"
```
